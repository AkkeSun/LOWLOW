=== Tag name

|===
| Tag name | Method | Description

| `XXX-page`
| `GET`
| `데이터를 페이지 형태로 가져옵니다`

| `XXX-get`
| `GET`
| `한 개 데이터를 가져옵니다`

| `XXX-create`
| `POST`
| `새로운 데이터를 등록합니다`

| `XXX-update`
| `UPDATE`
| `기존 데이터를 수정합니다`

| `XXX-delete`
| `DELETE`
| `기존 데이터를 삭제합니다`
|===
